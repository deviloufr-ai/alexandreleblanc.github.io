JobTrackr

Centralize, automate and pilot your job search — with AI as co-pilot

🌐 React web app + Vite 🤖 Claude Haiku (Anthropic) 📧 Gmail OAuth 📅 Google Calendar ☁️ Vercel · localStorage → Supabase v1.0 🦊 Firefox Extension
⚠️The Problem — Managing a job search without proper tools is chaos

After 20+ concurrent applications, every serious job seeker hits the same roadblocks:

Fragmentation — LinkedIn, Indeed, WTTJ, emails, networks: zero centralized hub.
Loss of context — Impossible to track where you are, who to follow up with, what was said.
Invisible emails — Rejections, confirmations and follow-ups drown in the inbox.
No priorities — What to do today? Who to call back? What's dead?
Rigid CV — Same CV sent everywhere, no adaptation to job descriptions.
Wrong tools — Heavy CRMs, unmanageable Excel beyond 30 rows.
🎯What is JobTrackr

Full-stack web application that centralizes application tracking and automates repetitive tasks with AI. A single dashboard — statuses, timelines, priorities, prep tools — automatically powered by Gmail sync. Built for the active job seeker managing 15 to 50+ concurrent applications who needs rigor without friction.

⚙️Features
📋Dashboard
Unified view of all applications. Filter by status/period/keyword. Multi-column sorting. 4 stat cards + activity sparklines.
📧Auto Gmail import
Google OAuth. 12 parallel requests. AI parses emails and creates or updates each application — zero manual entry. Auto-refresh on load.
🔄Statuses & timeline
10 business statuses. Auto-detect ATS rejections (Greenhouse, Lever, Ashby…). Complete per-app timeline with dates and sources.
⚡Next actions
Rule engine: applications to follow up on, imminent interviews, use-case deadlines. Differentiated urgent alerts, session-dismissable.
📅Calendar & meetings
Google Calendar integration (OAuth). Agenda widget, date grouping, today/tomorrow detection. View upcoming interviews.
📨Follow-up email
AI-generated contextualized follow-up draft. Direct send via Gmail if true recruiter detected — never to ATS no-reply addresses.
⭐STAR generator
3 contextualized STAR answers (Situation/Task/Action/Result) generated by AI. Interview prep in 1 click, quick copy per block.
📄Adaptive CV
Upload PDF. AI rewrites CV based on job description. Markdown editor + PDF export. Side-by-side before/after view.
📦Use case & deliverables
Technical test tracking: deadline, deliverables (GitHub, Notion, Figma, App…), item status, countdown. Auto-surfaces in urgent actions.
🎯Goals & KPIs
3 configurable KPIs: applications/week, response rate, interviews/month. Real-time progress bars.
🔍Job search
Integrated search via Adzuna API. Location + keyword filters. Direct add to applications from results.
🦊Firefox extension
Capture a job offer from any page. Analysis by Claude Vision. Pre-fill form via URL params. 1-click install.
🤖
AI layer — Claude Haiku (Anthropic) via Vercel Serverless proxy — API key never exposed client-side. AI operates at 6 levels:
Email parsing (batch 15) Status & company detection STAR generation Follow-up email draft Personalized CV rewriting Job offer analysis via extension (Vision)
